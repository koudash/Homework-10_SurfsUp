NOTE:
  1. API queries can be commited by typing the designated url in address bar.
  2. Alternatively, queries for the top three APIs can be simply made by clicking the designated hyperlink.
  3. The bottom two APIs require the replacement of highlighted keywords start and/or end with query dates in
    "yyyy-mm-dd" format if typed in address bar.
  4. An alternative query for the bottom two APIs is to directly select dates from drop-down menu and click "Search".
  5. Note that date entry must between "2010-01-01" and "2017-08-23", inclusive.
